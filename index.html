About

I'm a game programmer and iOS developer, living in São Paulo. In 2020, I'm finishing my graduation in game development at Mackenzie and the iOS developer program at the Apple Developer Academy | Mackenize.

I'm passionate about solving hard problems and learning about graphics programming.

I find very important to learn new things in a deep level of understanding before actually trying do build something with it. Even if takes longer, it will soon pay off.

I'm very strict about code robustness and maintainability. Specially in a game project because it requires a very high amount of iteration.

In my spare time I like to mess around with the guitar and make something new for my ears to enjoy.

Beard Love

This is my most recent project. It's a platformer in which you control a man looking for a woman who will accept his bearded love. I did the programming for all the character states and the state machine associated with it, tools for helping create the levels and more. I also did the music and the sound effects. It was all done in SpriteKit and it's still under development.

Super Music Cube

This is a personal project about an unusual and novel musical intrument. You can blace balls and arrows that guide them arround the cube. When a ball meets an arrow it plays the note associated with that cube's face. I used SceneKit for the graphics and AVAudioEngine to create a simple synthesizer

Super Brick Mania

This was a personal project meant as a way learn the Metal graphics API. It's a simple Breakout style game. All the graphics are made using Metal and MetalKit. It was challenging but I'm very glad I managed to do it and learned a lot about computer graphics in the way!

Junkyard Arena

This is a strategy game inspired by Into the Breach. It was all done using SpriteKit and was my first real attempt at making a complex game. I did the programming for the grid, character movement and interation, animations and more.

Nothing Weird

This was a personal project in which I wanted to make a nonsensical game. It's a point-and-click with one simple objetive: fry the potato! The idea is that the player than tries to click around and make some logical move but aways something unexpected happens instead. It was a fun little project made using UIKit.

Wack-a-Mole AR

This was a fun little project for learning about Augmented Reality and 3D modeling. It's a little wack-a-mole game where moles come out of a floating cube and the player has to physically move around it to hit them, earning points. I did all of the modeling and texturing and also helped with coding.
